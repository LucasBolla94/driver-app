# Notification Sound

Adicione um arquivo de som MP3 chamado `notification.mp3` nesta pasta.

## Recomendações:
- Formato: MP3
- Duração: 2-3 segundos
- Volume: Médio-Alto
- Tipo: Som de notificação/alerta (exemplo: sino, beep, etc.)

## Onde conseguir:
- https://freesound.org/ (sons gratuitos)
- https://mixkit.co/free-sound-effects/notification/
- Criar seu próprio som

## Fallback:
Se o arquivo não existir, o app tentará usar sons do sistema.
